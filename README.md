# Umbra

Umbra - Unity-проект, созданный на Unity `6000.4.5f1`.

## Требования

- Unity `6000.4.5f1` или совместимая версия Unity 6
- Git

## Как открыть проект

1. Склонируйте репозиторий.
2. Откройте корневую папку репозитория через Unity Hub.
3. Дождитесь, пока Unity восстановит пакеты из `Packages/manifest.json`.
4. Откройте одну из сцен в `Assets/Scenes`.

## Структура проекта

- `Assets/` - игровые ассеты, сцены, скрипты, UI, арт и настройки.
- `Packages/` - манифест Unity-пакетов и lock-файл.
- `ProjectSettings/` - настройки Unity-проекта.
